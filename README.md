# Flask Auction Site
An attempt at creating a flask app with Vue frontend for placing bids on items.

## Prerequisites

Make sure you have the following installed:

- Python 3.10+  
- Node.js 18+ and npm  
- Git

## Installation

### Backend
```bash
git clone https://github.com/Lumenality/flask-auction-project.git
cd flask-auction-project

python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Usage

### Start the backend
```bash
flask run
```

Then open the app in your browser at the local address shown in the terminal.
